<h1 align="center">
W&R Archive –
</h1>
<p align="left">
  <strong>это приложение разработанное под ОС Android, предназначенное для хранения гарантий и чеков, которое имеет удобные функции для помощи в отслеживании этих документов.</strong>
</p>

## О проекте

**W&R Archive Client** – это клиентская часть проекта. Серверная часть находится в репозитории **W-R-Archive-server**.

## Возможности приложения
**–** Создание, чтение, изменение и удалений гарантий и чеков
**–** Сохранение одной фотографии документа для каждой гарантии и чека
**–** Удобное отслеживание срока истечения гарантии
**–** Сортировка документов по дате
**–** Поиск по названию или описанию документа
**–** Возможность сохранения входа в аккаунт
**–** Возможность переключения цвета темы
**–** Приложение доступно на **Русском** и **Английском** языках
**–** Авторизация и регистрация

## Архитектура
В проекте применён архитектурный принцип Clean Architecture и паттерн MVVM.
